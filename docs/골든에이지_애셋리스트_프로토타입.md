## 문서 개요

프로토타입 개발에 필요한 모든 에셋을 정리한 문서입니다.
각 에셋은 **필수/선택**으로 구분되어 있으며, **무료/유료** 소스도 함께 안내합니다.

### 범례
- 🔴 **필수**: 프로토타입 완성에 반드시 필요
- 🟡 **권장**: 있으면 좋지만 대체 가능
- 🟢 **선택**: 폴리싱 단계에서 추가

### 제작 vs 구매 기준
- **직접 제작**: 핵심 캐릭터, 독창적 아이템
- **구매/무료**: 배경, 소품, 범용 이펙트

### 3.4 Mixamo 활용 가이드

#### Mixamo란?
Adobe에서 제공하는 **무료** 3D 캐릭터 애니메이션 라이브러리입니다.
수천 개의 애니메이션을 FBX 형식으로 다운로드할 수 있습니다.

#### 사용 방법

```
1. Mixamo 접속
   - URL: https://www.mixamo.com
   - Adobe 계정으로 로그인 (무료)

2. 캐릭터 업로드 (선택)
   - 자체 캐릭터 모델 업로드 가능
   - 또는 Mixamo 기본 캐릭터 사용

3. 애니메이션 검색
   - 검색창에 키워드 입력 (예: "idle", "walk", "punch")
   - 미리보기로 확인

4. 다운로드 설정
   - Format: FBX for Unity (.fbx)
   - Skin: With Skin (캐릭터 포함) 또는 Without Skin (애니메이션만)
   - Frames per Second: 30
   - Keyframe Reduction: None (정밀) 또는 Uniform (용량 절감)

5. Unity Import
   - FBX를 Assets 폴더로 드래그
   - Rig 탭: Animation Type → Humanoid
   - Animation 탭: Loop Time 체크 (루프 애니메이션인 경우)
```

#### 권장 Mixamo 애니메이션

| 용도 | Mixamo 검색어 | 비고 |
|------|-------------|------|
| 대기 | "Idle", "Breathing Idle" | 남성/여성 버전 |
| 걷기 | "Walking", "Walk Strafe" | 방향별 다양 |
| 달리기 | "Running", "Sprint" | 속도별 선택 |
| 근접 공격 | "Punch", "Hook", "Jab" | 콤보용 여러 개 |
| 사격 | "Pistol Idle", "Pistol Fire" | 총기 애니메이션 |
| 피격 | "Hit Reaction", "Big Hit" | 강약 구분 |
| 사망 | "Death", "Dying" | 여러 스타일 |

#### 주의사항

```
- 상업적 사용 가능 (라이선스 확인)
- 캐릭터 모델은 Mixamo 외부 반출 시 라이선스 확인 필요
- 애니메이션만 다운로드하면 자체 캐릭터에 적용 가능
- Humanoid 리깅된 캐릭터만 호환
```
