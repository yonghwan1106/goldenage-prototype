## 6. 에셋 저장 위치

```
unity/Assets/_Project/
├── Animations/
│   ├── Downloaded/          ← Mixamo 다운로드
│   │   ├── Player/
│   │   ├── NPC/
│   │   └── Enemy/
│   └── Controllers/
├── Audio/
│   ├── BGM/
│   ├── SFX/
│   └── Downloaded/          ← Freesound 다운로드
├── Models/
│   └── Downloaded/          ← Asset Store 다운로드
├── Textures/
│   └── Downloaded/          ← AmbientCG 다운로드
└── ThirdParty/              ← Asset Store 패키지
```
